## 목적

## 진행

- 개인 프로젝트

## 환경

- 프레임워크: Spring

## 설명
### 페이지 접속
- 127.0.0.0:8080/home으로 접속합니다.

상단의 네비게이션 바를 통해 다른 페이지로 이동할 수 있습니다.</br>
Test, Home은 /home으로, Articles는 /articles로 이동합니다.</br>
